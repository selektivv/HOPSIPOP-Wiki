# Mining World Access

Reach Atom [rank](../ranks.md), then run `/cw` and select Enter [Capacity](../capacity.md) World. The server sends you to a random safe location inside the border.

The `/cw` menu also contains mining upgrades, [Notifications](notifications.md), and [Chunk Drill](chunk-drills.md) management. `/shortcuts` provides another entrance after Atom is reached.

## Rank Haste

| Rank | Effect |
| --- | --- |
| Atom | Haste I |
| Nova | Haste II |
| Quasar or higher | Haste III |

Haste is removed when you leave the world.

## Before Entering

- Store valuables safely.
- Check the reset timer.
- Remember that PvP is enabled.
- Leave free Master Chest [Capacity](../capacity.md) for drill output.
