# Access and Getting Started

## Unlocking the World

[Capacity](../capacity.md) World access begins at Atom [rank](../ranks.md). Open `/perks` and select [Rank Overview](../ranks.md) to see your current [Capacity](../capacity.md), the live Atom requirement, and your progress toward later [ranks](../ranks.md).

[Rank](../ranks.md) requirements can change as the server evolves, so the in-game menu is the reliable source for current thresholds.

## Entering the World

1. Run `/cw`.
2. Select Enter [Capacity](../capacity.md) World.
3. The server sends you to a random safe location inside the world border.
4. Start mining eligible blocks and follow the [counter](counters.md) shown above your hotbar.

The `/shortcuts` menu also provides [Capacity](../capacity.md) World access after you reach Atom [rank](../ranks.md).

## What the Capacity World Menu Contains

| Menu option | Purpose |
| --- | --- |
| Enter [Capacity](../capacity.md) World | Teleports you to a random location in the mining world. |
| [Counter Reduction](counter-reduction.md) | Reduces the number of eligible blocks required for each [Capacity](../capacity.md) reward. |
| [Lucky Break](lucky-break.md) | Gives manually mined blocks a chance to add extra [counter](counters.md) progress. |
| [Co-op Size](coop.md#requirements) | Increases the number of players allowed in your [co-op](coop.md). |
| [Chunk Drill](chunk-drills.md) | Unlocks automated chunk excavation or opens drill management after purchase. |
| [Notification Interval](boosts-and-notifications.md#notification-interval) | Controls how often completed-goal messages appear. |

## Rank Benefits Inside the World

Your [rank](../ranks.md) automatically applies mining Haste while you are in the [Capacity](../capacity.md) World:

| Rank | Haste |
| --- | --- |
| Atom | Haste I |
| Nova | Haste II |
| Quasar and above | Haste III |

Haste is removed when you leave the [Capacity](../capacity.md) World. Skill mana abilities also last ten times longer while used inside the world.

## Before You Begin

- Store valuables safely, even though Keep Inventory is enabled.
- Check the reset countdown on the [Capacity](../capacity.md) World scoreboard.
- Remember that PvP is enabled.
- Do not build anything you expect to keep; terrain and structures are temporary.
- Leave free [Master Chest](../master-chest/README.md) [capacity](../capacity.md) before using [Chunk Drills](chunk-drills.md).
