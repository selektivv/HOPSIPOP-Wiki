# Mining and Progression: Capacity World

The [Capacity](../capacity.md) World is the repeatable mining branch of HOPSIPOP's connected storage and progression system. Mining [counter](counters.md) goals award [Capacity](../capacity.md), [Capacity](../capacity.md) expands the Master Chest network and [ranks](../ranks.md), and [Chunk Drill](chunk-drills.md) output returns to storage.

Read the [system overview](../system-overview.md) to see the complete loop, or read [Capacity](../capacity.md) before spending on upgrades.

## What the Mining World Does

- Awards [Capacity](../capacity.md) for repeated mining goals.
- Gives [rank](../ranks.md)-based Haste while you are inside the world.
- Supports shared mining progress through [co-op](coop.md) groups.
- Offers permanent [Counter Reduction](counter-reduction.md), [Lucky Break](lucky-break.md), [Co-op Size](coop.md#requirements), and [Chunk Drill](chunk-drills.md) unlocks.
- Supports temporary [Counter Boost](boosts-and-notifications.md) vouchers that triple your personal reward.
- Lets [Chunk Drills](chunk-drills.md) excavate complete chunks and transfer produced drops to Master Chest storage.
- Recreates the world every ten hours so fresh terrain remains available.

## How It Connects to Storage

- Atom [rank](../ranks.md) is required before entering.
- Completed mining goals add [Capacity](../capacity.md) immediately.
- Added [Capacity](../capacity.md) expands storage space and [rank](../ranks.md) progress.
- Mining upgrades and claims spend that same [Capacity](../capacity.md).
- [Chunk Drill](chunk-drills.md) drops and returning drill items are transferred to the owner's Master Chest network.
- A full or over-limit storage network can prevent drill transfers from fitting safely.

## Main Commands

| Command | Purpose |
| --- | --- |
| `/cw` | Open the [Capacity](../capacity.md) World menu. |
| `/perks` | Check your live [rank](../ranks.md), requirements, and benefits. |
| `/coop <player>` | Invite an online player to your mining [co-op](coop.md). |
| `/coop decline <player>` | Decline a pending invitation. |
| `/coop leave` | Leave your current [co-op](coop.md). |
| `/chunkdrill` | Open [Chunk Drill](chunk-drills.md) management after unlocking it. |
| `/shortcuts` | Open shortcuts for storage and connected systems. |

## The Most Important Spending Rule

Mining upgrades spend actual [Capacity](../capacity.md). Before buying one, make sure the remaining value still covers stored items and preserves any [rank](../ranks.md) benefits you need.

Spending never deletes existing stored items, but new deposits can stop until the stored total is below the reduced limit.
