# Counter Reduction

Counter Reduction is a permanent mining-world upgrade that lowers the number of eligible blocks required to complete one [counter](counters.md) goal. The default goal is ten blocks; buying every level reduces it to one.

## Buying the Upgrade

Open `/cw` and select Counter Reduction. Each purchase immediately spends [Capacity](../capacity.md), unlocks the next lower goal, and cannot be refunded.

| New block goal | Upgrade cost |
| --- | ---: |
| 9 blocks | 100,000 [Capacity](../capacity.md) |
| 8 blocks | 300,000 [Capacity](../capacity.md) |
| 7 blocks | 900,000 [Capacity](../capacity.md) |
| 6 blocks | 2,700,000 [Capacity](../capacity.md) |
| 5 blocks | 8,100,000 [Capacity](../capacity.md) |
| 4 blocks | 24,300,000 [Capacity](../capacity.md) |
| 3 blocks | 72,900,000 [Capacity](../capacity.md) |
| 2 blocks | 218,700,000 [Capacity](../capacity.md) |
| 1 block | 656,100,000 [Capacity](../capacity.md) |

Each row is the price of that individual step, not a cumulative total.

## What It Affects

The lower goal applies to eligible blocks mined manually and blocks processed by [Chunk Drills](chunk-drills.md). It does not increase the reward for a completed goal; it makes goals finish sooner.

[Lucky Break](lucky-break.md) and a [Counter Boost](boosts-and-notifications.md) are separate effects. [Lucky Break](lucky-break.md) can add extra manual progress, while a boost changes the personal reward after the goal is completed.

## Co-op Behavior

When a new [co-op](coop.md) is formed, the player with the better Counter Reduction becomes the initial group owner. Active members use that owner's block goal. Adding a member with a better personal goal later does not automatically change the owner.

## Before Buying

The purchase reduces the same [Capacity](../capacity.md) used for storage and [ranks](../ranks.md). Check free Master Chest space and [rank](../ranks.md) progress before buying an expensive level.
