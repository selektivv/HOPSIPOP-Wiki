# Cow Attraction Block

The Cow Attraction Block guides nearby Cows toward one fixed point. It can be crafted at the [Tinkerer Table](../tinkerer-table.md) without a rank requirement.

## Tinkerer Table Recipe

Place the following items in the table's 3-by-3 ingredient grid:

| | | |
|---|---|---|
| Leather | Hay Bale | Leather |
| Hay Bale | Emerald | Hay Bale |
| Leather | Bell | Leather |

When the pattern is correct, the Cow Attraction Block appears in the result slot. Click it to consume all nine ingredients and receive the block.

## Effect

Once every second, Cows within **20 blocks** are told to pathfind toward the placed block. The effect runs only while its chunk is loaded.

The block does not teleport Cows or override terrain and pathfinding. Build a clear route if the animals need to reach a pen or collection area.

## Placement Rules

- Any player can place and break the block.
- Up to four Cow Attraction Blocks can be placed on the server at a time.
- It cannot be placed in the [Capacity World](../../capacity-world/README.md).
- Breaking it returns the special Cow Attraction Block item.
