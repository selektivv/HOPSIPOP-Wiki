# Chicken Attraction Block

The Chicken Attraction Block guides nearby Chickens toward one fixed point. It can be crafted at the [Tinkerer Table](../tinkerer-table.md) without a rank requirement.

## Tinkerer Table Recipe

Place the following items in the table's 3-by-3 ingredient grid:

| | | |
|---|---|---|
| Copper Block | Feather | Copper Block |
| Egg | Redstone Block | Egg |
| Iron Block | Iron Block | Iron Block |

When the pattern is correct, the Chicken Attraction Block appears in the result slot. Click it to consume all nine ingredients and receive the block.

## Effect

Once every second, Chickens within **20 blocks** are told to pathfind toward the placed block. The effect runs only while its chunk is loaded.

The block does not teleport Chickens or override terrain and pathfinding. Build a clear route if the animals need to reach a pen or collection area.

## Placement Rules

- Any player can place and break the block.
- Up to four Chicken Attraction Blocks can be placed on the server at a time.
- It cannot be placed in the [Capacity World](../../capacity-world/README.md).
- Breaking it returns the special Chicken Attraction Block item.
