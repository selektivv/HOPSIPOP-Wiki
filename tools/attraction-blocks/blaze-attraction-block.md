# Blaze Attraction Block

The Blaze Attraction Block guides nearby Blazes toward one fixed point. It can be crafted at the [Tinkerer Table](../tinkerer-table.md) without a rank requirement.

## Tinkerer Table Recipe

Place the following items in the table's 3-by-3 ingredient grid:

| | | |
|---|---|---|
| Blaze Rod | Magma Cream | Blaze Rod |
| Magma Cream | Respawn Anchor | Magma Cream |
| Blaze Rod | Magma Cream | Blaze Rod |

When the pattern is correct, the Blaze Attraction Block appears in the result slot. Click it to consume all nine ingredients and receive the block.

## Effect

Once every second, Blazes within **20 blocks** are told to pathfind toward the placed block. The effect runs only while its chunk is loaded.

The block does not make Blazes harmless: it does not suppress attacks, change their targeting, or teleport them through obstacles. Build the surrounding route and containment area accordingly.

## Placement Rules

- Any player can place and break the block.
- Up to four Blaze Attraction Blocks can be placed on the server at a time.
- It cannot be placed in the [Capacity World](../../capacity-world/README.md).
- Breaking it returns the special Blaze Attraction Block item.
