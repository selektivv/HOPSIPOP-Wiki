# Lava Sponge

A Lava Sponge removes nearby lava when placed.

## Crafting

Combine one Sponge with one Magma Cream. The recipe is shapeless.

![Lava Sponge crafting recipe](../assets/lava-sponge-crafting.png)

## Using the Sponge

Place it near lava to remove up to 64 lava blocks within six blocks. Break it afterward to recover the special Lava Sponge.
