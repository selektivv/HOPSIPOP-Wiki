## The Progression Loop

1. Complete early progression to unlock the Master Chest storage network.
2. Store and manage items through `/mc`.
3. Earn [Capacity](capacity.md) to expand storage and advance through ranks.
4. Reach Atom rank and use the [Capacity](capacity.md) World as a repeatable [Capacity](capacity.md) source.
5. Spend [Capacity](capacity.md) carefully on claims and permanent mining upgrades.
6. Use OmniSync, automation, hoppers, co-op mining, and Chunk Drills to connect the systems together.

## Quick Start

1. Equip a full set of iron armor at least once.
2. Complete the Verdant dungeon to unlock your storage network.
3. Run `/mc` and shift-click items into the menu.
4. Run `/roadmap` to see the next progression requirement.
5. Run `/perks` to check [Capacity](capacity.md)-rank progress.
6. After reaching Atom, run `/cw` to open the mining-world menu.
