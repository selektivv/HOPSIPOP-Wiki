# Sharing Networks

Sharing lets trusted players browse, deposit into, and withdraw from your Master Chest network.

## Granting Access

Run `/mc invite <player>` to give a player access to your network.

Run `/mc uninvite <player>` to remove that access.

An invitation is an access grant, not a one-time request. Only invite players you trust with the full contents of your network.

## Opening a Shared Network

Run `/mc shared` to see every player who currently shares a network with you. Select a player's head to browse that network.

You may also open a specific shared network with `/mc shared <player>`.

A placed Master Chest belonging to another player opens for you only if that owner has invited you.

## Deposits and Withdrawals

The normal transfer controls work in a shared network. This means an invited player can both add and remove items. The shared network uses the owner's capacity and automation settings.

Always check whose network is open before shift-clicking a large inventory into it.

## Removing Access

Uninviting a player prevents future access through the shared menu and placed access points. It does not automatically return items that the player previously deposited or withdrew.

If another player's network disappears from your Shared Chests menu, the owner has removed your access.
