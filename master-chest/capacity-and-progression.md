# Progression and [Capacity](../capacity.md) Ranks

[Capacity](../capacity.md) ranks connect storage growth to feature unlocks and server benefits. Read [Capacity](../capacity.md) first for the full explanation of earning, spending, storage limits, claims, and upgrade effects.

## Checking Progress

- Run `/playtime` to see your storage limit, playtime, and first join date.
- Open the storage network and hover over Shortcuts to compare total items with free space.
- Open `/perks` to view rank requirements and benefits.
- Open `/roadmap` to view feature-progression requirements.

Playtime is recorded and displayed, but it does not currently add [Capacity](../capacity.md) by itself.

## Earning Progress

[Capacity](../capacity.md) is awarded by HOPSIPOP activities and progression. Current sources include dungeon milestones, eligible Masterlands mob kills, [mining-counter goals](../capacity-world/mining-and-rewards.md), voting rewards, daily-event prizes, Vote Crate rewards, and other server rewards.

The scoreboard and reward messages show changes as they happen. Use the in-game roadmap and menus as the live source for your next goal.

## Feature Unlocks

| Progression step | Connected benefit |
| --- | --- |
| Complete Verdant | Unlock the Master Chest storage network. |
| Complete Mobmon | Unlock OmniSync crafting. |
| Complete Stonemason | Unlock Stone Cut automation. |
| Complete Crafterlands | Unlock Craft automation and progress toward Furnacity. |
| Reach Infinity wave 21 | Unlock the portable Workbench. |
| Reach Infinity wave 50 | Unlock Ender Chest access. |
| Complete Furnacity | Unlock Smelt automation. |
| Reach Atom rank | Unlock the [Capacity](../capacity.md) World and its repeatable mining progression. |

The armor steps shown in `/roadmap` are also required parts of the progression sequence.

## [Capacity](../capacity.md) Ranks

The current rank order is:

Atom, Nova, Quasar, Singularity, Void, Cosmic, Celestial, Divine, Primordial, Multiversal, Omniversal, and Apex.

Open `/perks`, then Rank Overview, to see the live [Capacity](../capacity.md) requirement for each rank. Left-click a rank to track it with a boss bar; right-click a rank to stop tracking.

Rank requirements can be adjusted as the server evolves, so the in-game Rank Overview is more reliable than memorizing a number from a guide.
